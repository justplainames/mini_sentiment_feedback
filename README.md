# Mini Sentiment Feedback

The goal is to complete the code within the stipulated amount of time. This app is deployed on an EC2 instance and served via Nginx accessible here - http://54.255.172.66

## Install

If nothing is installed yet:

1. Download and install `Node.js` version `20.19+` or `22.12+` from the official Node.js website. https://nodejs.org/en/download
2. Check that Node and npm are installed:

```bash
node -v
npm -v
```

## Run

```bash
npm install
npm run dev
```

By default it should run here

```text
http://localhost:5173
```
